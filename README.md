# Hospital Management System (C++)

## Overview
A basic hospital management system developed in C++ during my first semester at UMT. The project demonstrates core programming concepts including procedural programming, file handling, and basic data management.

## Features
- Patient record management (add, view, update, delete)
- Staff and appointment tracking
- Simple command-line interface for interaction
- Basic data persistence using files

## Technologies
- C++ (OOP concepts planned for future versions)
- File Handling for persistent storage
